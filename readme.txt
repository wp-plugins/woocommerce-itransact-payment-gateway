This is a Plugin to integrate iTransact Payment Gateway with WooCoommerce.

== Description ==

You just need to enter the Vendor ID obtained from iTransact Gateway and everything is ready. Users will be redirected to iTransact Gateway from WooCommerce to enter their payment information.

Features

* Simply activate the plugin, enter Vendor ID and you are ready to go.


== Installation ==

Automatic WordPress Installation

1. Login to your WordPress Site
2. Under the 'Plugins' sidebar tab, click ‘Add New’
3. Search for ‘WooCommerce iTransact Payment Gateway’
4. Install and Activate the Plugin
5. Navigate to 'WooCommerce > Settings' and click on 'Checkout' tab
6. Enter the 'Vendor ID' you obtained from iTransact Gateway (please see below for the instructions)
* Login in to iTransact Gateway
* Navigate through 'Control Panel > Account Settings'
* Copy 'Order Form UID' from 'Advanced Features' section

Manual Installation

1. Download the latest version of the plugin from 'WooCommerce iTransact Payment Gateway' WordPress Plugin page.
2. Uncompress the file
3. Upload the uncompressed directory to ‘/wp-content/plugins/’ via FTP
4. Active the plugin from your WordPress backend ‘Plugins > Installed Plugins’
5. Navigate to 'WooCommerce > Settings' and click on 'Checkout' tab
6. Enter the 'Vendor ID' you obtained from iTransact Gateway

== Frequently Asked Questions ==

= How to obtain 'Vendor ID' from iTransact Payment Gateway? =

Just login into iTransact Gateway using your credentials. Navigate through 'Control Panel' and then 'Account Settings'. Copy 'Order Form UID' value from 'Advanced Features' section.
